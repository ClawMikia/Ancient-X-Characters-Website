# Ancient X Characters Website

A dark fantasy character showcase website featuring Ancient X warriors, villains, and legendary figures.

## Project Structure

```
├── index.html          # Homepage with hero banner and featured characters
├── about.html          # About / lore page
├── characters.html     # Character gallery with filtering and detail viewer
├── creator.html        # Creator profile page
├── style.css           # Global stylesheet (1339 lines)
├── characters.js       # Character data and gallery logic (655 lines)
├── main.js             # Homepage scripts
├── logo.png            # Site logo
├── background.png      # Hero background image
└── assets/Characters/  # Character artwork by category
    ├── Ancient X Fighters/
    ├── Villains/
    │   ├── Megahforce of Destruction/
    │   └── The Powerful Force of Destruction/
```

## Features

- Fixed responsive navbar with mobile hamburger menu
- Animated hero banner with particle effects
- Category-tabbed character gallery (fighters, villains, powerful)
- Interactive character detail panel with bio, stats, and story
- Expandable story cards
- Lazy-loaded character artwork with fallback placeholders
- Dark fantasy gold/crimson theme

## Character Categories

- **Ancient X Fighters** — Warriors like Alice, Garrett, Lloyd, Vladimir
- **Megahforce of Destruction** — Armourgeddon, Cyrobranxonn, Phoenomoux, etc.
- **The Powerful Force of Destruction** — Astermm, Melachon, Mortex, Zhekki, etc.

## Usage

Open `index.html` in a browser. The site is static with no build step required.
